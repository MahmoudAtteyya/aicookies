🚀 Quick Start

All endpoints are OpenAI-compatible. Replace your base_url and use any model slug below.

POST https://aicookies.elliaa.com/v1/{model-slug}/chat/completions

cURL

curl -X POST https://aicookies.elliaa.com/v1/mistral-small/chat/completions \
  -H "Content-Type: application/json" \
  -d '{"model":"mistral-small","messages":[{"role":"user","content":"Hello!"}]}'

Python (OpenAI SDK)

from openai import OpenAI
client = OpenAI(base_url="https://aicookies.elliaa.com/v1", api_key="x")
response = client.chat.completions.create(
    model="mistral-small",
    messages=[{"role":"user","content":"Hello!"}])
print(response.choices[0].message.content)

Node.js / TypeScript

const res = await fetch("https://aicookies.elliaa.com/v1/mistral-small/chat/completions", {
  method:"POST", headers:{"Content-Type":"application/json"},
  body: JSON.stringify({model:"mistral-small", messages:[{role:"user",content:"Hello!"}]})
});
console.log((await res.json()).choices[0].message.content);

🔄 Smart Key Rotation

  1. Request → least-used active key → real provider API
  2. 429 Rate Limit → rotate to next key (free providers recover)
  3. 402 Payment Required → Fireworks: 💀 DEAD permanently | Free: temporarily disabled
  4. 401/403 Auth Error → disabled after 10 consecutive errors
  5. Response includes X-Proxy-Provider, X-Proxy-Key, X-Proxy-Latency headers
